Lower-case hexadecimal encoding

This encoding is a static version of:

assert_eq!(HEXLOWER, &Builder::new(b"0123456789abcdef").no_pad().unwrap());

Examples

use data_encoding::HEXLOWER;
let deadbeef = vec![0xde, 0xad, 0xbe, 0xef];
